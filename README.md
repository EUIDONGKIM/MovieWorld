### 개발 환경
#### 운영체제(OS) 및 버전
 - window 10
#### Front End
  - CSS
  - JavaScript
  - JQuery 3.6.0
  - Bootstrap5
#### BACK END
  - Java 1.8
  - WAS(Web-Application-Server): Apache Tomcat 9.0

#### Database
  - RDBMS: Oracle database 11g

#### Developer tool
 - Spring
 - Oracle: SQL-Developer
 - VScode
 

#### api
 - 다음 우편번호 서비스 
 - 카카오지도
 - 카카오페이
